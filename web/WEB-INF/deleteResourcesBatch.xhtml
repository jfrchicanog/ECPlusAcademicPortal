Eliminar recursos

Instrucciones

  1. Seleccione la lista de palabras
  2. Introduzca en el área de texto la lista de palabras para aliminar sus recursos.
  3. Para cada palabra debe usar el formato "<palabra>:<id>". La palabra o el id pueden estar ausentes, pero no ambos.
  4. Pulse el botón para eliminar los pictogramas y las fotos.

Aclaraciones

  • Si una palabra no casa con ninguna palabra de la lista no hace nada.
  • Si varias palabras coinciden on la de la lista, se elimina los recursos en todos.

Ejemplo

Abuela:15
:17
Agua
